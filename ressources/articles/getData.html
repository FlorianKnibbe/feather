• Les données affichées sont issues des paramètres GET passés dans l'attribut data-view-article à la suite de l'id de l'article et stocké par Feather dans la variable Feather.get. L'id de l'article est #getData, la valeur de l'attribut data-view-article est donc #getData?cle1=valeur1&cle2=valeur2&cle3=Ma+valeur+3

  • Les différentes données stocké en javascript peuvent être injectées automatiquement dans une page de votre application grâce au gestionnaire de template de Feather.

    Si vous placez dans votre DOM une balise du type {\{XXXX}\} elle sera remplacée par la valeur correspondante si celle-ci existe, sinon elle sera remplacée par du vide.

  • Exemple de template avec get :

    Ma valeur 1 : {{Feather.get.cle1}}

    Ma valeur 2 : {{Feather.get.cle2}}

    Ma valeur 3 : {{Feather.get.cle3}}

    Ma valeur 4 qui n'existe pas : {{Feather.get.cle4}}
